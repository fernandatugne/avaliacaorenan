    Como hospedar um Site Estático no GitHub Pages?

    O GitHub Pages é a forma gratuita de hospedar um site estático.

    Hospedando um Site Estático

    • Inicie um repositório novo no GitHub com o username.github.io (onde o username é o nome do seu usuário no GitHub ou da sua organização).
    • Clone o repositório para sua máquina local usando o Git.
    • Para que o conteúdo seja hospedado diretamente pelo GitHub Pages é necessário que ele esteja no branch master do repositório.
    • Coloque seus arquivos no repositório.
    • Comite e faça o push das alterações para o repositório no GitHub.

      • Pronto, agora voce tem um site estático no GitHub.

    Hospedando um Site Estático em React

    • Crie um projeto React
    • Crie um repositório no Github e configure-o ao seu projeto.
    • Instale o gh-pages como dependência de desenvolvimento no projeto
    • Adicione propriedades no package.json
    • Configure o arquivo de rotas da aplicação
    • Dê o comando de deploy
    • Após o processo de deploy ser concluído, a sua aplicação já estará no ar.
